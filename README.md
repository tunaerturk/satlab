## ✨ Features

- 📚 **Smart Question Bank**  
  Filter questions by domain, difficulty, and topic for targeted practice.

- 👁️ **Advanced Question Experience**  
  Built-in timer, interactive tooltips, and instant feedback system.

- 📊 **Progress Analytics Dashboard**  
  Visualize your daily performance and identify weak areas.

- 🌓 **Seamless Dark Mode**  
  Clean, modern UI with full dark mode support.

- 📱 **PWA Ready**  
  Installable on mobile and works offline for uninterrupted study.

- ⚡ **Powered by SvelteKit 5**  
  Lightning-fast performance with modern web technologies.

## 🚀 Getting Started

Get up and running locally in just a few steps:

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- [npm](https://www.npmjs.com/) (usually comes with Node.js)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/tunaerturk/satlab.git
   ```
2. Navigate to the project directory:
   ```bash
   cd satlab
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development

Start the development server:

```bash
npm run dev
```
